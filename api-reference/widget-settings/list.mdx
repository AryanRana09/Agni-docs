---
title: "List Widget Settings"
openapi: "GET /api/v1/widget-settings"
description: "List all widget settings records."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object[]` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Widget settings list",
  "data": []
}
```

</ResponseExample>
