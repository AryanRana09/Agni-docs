---
title: "Update Widget Settings"
openapi: "PATCH /api/v1/widget-settings/{id}"
description: "Update one widget settings record."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### id

`integer` required

Widget settings id.

### Body

`application/json`

#### bot_name

`string`

#### bot_auto_start

`boolean`

#### bot_position

`string`

#### language

`string`

#### custom_form_fields

`object`

#### theme

`object`

<RequestExample>

```json
{
  "bot_name": "Updated Bot",
  "bot_auto_start": false,
  "bot_position": "BOT_POSITION_BOTTOM_LEFT",
  "language": "en",
  "custom_form_fields": {},
  "theme": {}
}
```

</RequestExample>

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`integer` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Widget updated",
  "data": {
    "id": 1
  }
}
```

</ResponseExample>
