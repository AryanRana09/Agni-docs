---
title: "Get Widget Settings"
openapi: "GET /api/v1/widget-settings/{id}"
description: "Get one widget settings record by id."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### id

`integer` required

Widget settings id.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`integer` required

#### data.bot_name

`string` required

#### data.bot_auto_start

`boolean` required

#### data.bot_position

`string` required

#### data.custom_form_fields

`object`

#### data.theme

`object`

#### data.language

`string`

#### data.is_active

`boolean` required

#### data.created_at

`string` required

ISO 8601 timestamp.

#### data.updated_at

`string` required

ISO 8601 timestamp.
