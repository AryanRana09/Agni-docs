---
title: "Create Widget Settings"
openapi: "POST /api/v1/widget-settings"
description: "Create widget settings used by your web widget."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Body

`application/json`

#### bot_name

`string` required

Widget bot name.

#### bot_auto_start

`boolean`

Auto-starts the widget when page loads.

#### bot_position

`string`

Widget position.

#### language

`string`

Widget language code.

#### custom_form_fields

`object`

Custom pre-chat form fields.

#### theme

`object`

Widget theme settings.

<RequestExample>

```json
{
  "bot_name": "Support Bot",
  "bot_auto_start": true,
  "bot_position": "BOT_POSITION_BOTTOM_RIGHT",
  "language": "en",
  "custom_form_fields": {},
  "theme": {}
}
```

</RequestExample>

### Response

#### 201 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`integer` required

<ResponseExample>

```json 201
{
  "success": true,
  "message": "Widget created",
  "data": {
    "id": 1
  }
}
```

</ResponseExample>
