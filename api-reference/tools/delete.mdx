---
title: "Delete Tool"
openapi: "DELETE /api/v1/tools/{id}"
description: "Delete one tool by id."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### id

`integer` required

Tool id.

### Query parameters

#### organization_id

`string` required

Organization id.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Tool deleted"
}
```

</ResponseExample>
