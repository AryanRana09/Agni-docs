---
title: "List Tools"
openapi: "GET /api/v1/tools"
description: "List tools in an organization with pagination."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Query parameters

#### organization_id

`string` required

Organization id.

#### limit

`integer`

Number of records to return.

#### offset

`integer`

Number of records to skip.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object[]` required

#### meta

`object` required

#### meta.total

`integer` required

#### meta.limit

`integer` required

#### meta.offset

`integer` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Tools fetched",
  "data": [],
  "meta": {
    "total": 1,
    "limit": 10,
    "offset": 0
  }
}
```

</ResponseExample>
