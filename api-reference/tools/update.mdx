---
title: "Update Tool"
openapi: "PATCH /api/v1/tools/{id}"
description: "Update an existing tool."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### id

`integer` required

Tool id.

### Body

`application/json`

#### organization_id

`string` required

Organization id.

#### name

`string`

#### description

`string`

#### type

`string`

#### definition

`object`

<RequestExample>

```json
{
  "organization_id": "org_123",
  "name": "Updated Tool",
  "description": "Updated description",
  "type": "api",
  "definition": {}
}
```

</RequestExample>

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`integer` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Tool updated",
  "data": {
    "id": 1
  }
}
```

</ResponseExample>
