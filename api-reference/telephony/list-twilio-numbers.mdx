### Overview

Use this endpoint to **list phone numbers owned by your connected telephony provider** that are available to use with your account.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `GET`  
- **Path**: `/twilio-owned-numbers`

### Successful response (200)

On success, the API returns a list of numbers owned by the telephony provider.

#### Top-level fields

| Field     | Type                           | Description                                   |
| --------- | ------------------------------ | --------------------------------------------- |
| `success` | boolean                        | Indicates whether the request was successful. |
| `message` | string                         | Human-readable message about the operation.   |
| `data`    | array\<ProviderOwnedNumber>    | List of provider-owned numbers.               |

#### `ProviderOwnedNumber` object

This object corresponds to the `TwilioOwnedNumber` message in the service definition.

| Field           | Type    | Description                                             | Example                  |
| -------------- | ------- | ------------------------------------------------------- | ------------------------ |
| `sid`          | string  | Provider-side identifier for the phone number.         | `"PNXXXXXXXXXXXXXXXX"`   |
| `phone_number` | string  | Phone number in E.164 format.                           | `"+14157774444"`         |
| `friendly_name`| string  | Human-readable label for the number.                   | `"Support Line"`         |
| `status`       | string  | Current status of the number with the provider.        | `"active"`               |
| `date_created` | string  | Creation timestamp at the provider.                     | `"2024-01-01T00:00:00Z"` |
| `voice`        | boolean | Whether the number supports voice.                      | `true`                   |
| `sms`          | boolean | Whether the number supports SMS.                        | `true`                   |
| `mms`          | boolean | Whether the number supports MMS.                        | `false`                  |
