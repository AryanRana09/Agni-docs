### Overview

Use this endpoint to **purchase a phone number** that you have previously discovered via the List Available Numbers API.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `POST`  
- **Path**: `/phone-numbers/buy`

### Request body

Send a JSON object with the following fields.

| Field                        | Type    | Required | Description                                                                 | Example         |
| --------------------------- | ------- | -------- | --------------------------------------------------------------------------- | --------------- |
| `phone_number`              | string  | Yes      | Phone number to purchase in E.164 format.                                  | `"+14157774444"` |
| `price`                     | number  | Yes      | One-time purchase price for the number.                                    | `1.0`           |
| `per_minute_price_inbound`  | int32   | Yes      | Per-minute price (in minor units) for inbound calls to this number.        | `1`             |
| `per_minute_price_outbound` | int32   | Yes      | Per-minute price (in minor units) for outbound calls from this number.     | `2`             |

<RequestExample>

```bash 201 - Buy phone number
curl --request POST \
  --url https://api.ravan.ai/api/v1/phone-numbers/buy \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "phone_number": "+14157774444",
    "price": 1.0,
    "per_minute_price_inbound": 1,
    "per_minute_price_outbound": 2
  }'
```

</RequestExample>

### Successful response (201)

On success, the API returns the purchased phone number record.

#### Top-level fields

| Field     | Type                    | Description                                   |
| --------- | ----------------------- | --------------------------------------------- |
| `success` | boolean                 | Indicates whether the request was successful. |
| `message` | string                  | Human-readable message about the operation.   |
| `data`    | PurchasedPhoneNumber    | Purchased phone number details.               |

#### `PurchasedPhoneNumber` object

| Field           | Type    | Description                                        | Example            |
| -------------- | ------- | -------------------------------------------------- | ------------------ |
| `phone_number` | string  | Purchased phone number in E.164 format.           | `"+14157774444"`   |
| `price`        | number  | One-time purchase price for the number.           | `1.0`              |
| `status`       | string  | Status of the purchased number (for example, `"active"`). | `"active"` |
| `purchased_at` | string  | ISO 8601 timestamp when the number was purchased. | `"2024-01-01T00:00:00Z"` |

<ResponseExample>

```json 201 - Created
{
  "success": true,
  "message": "Phone number purchased successfully.",
  "data": {
    "phone_number": "+14157774444",
    "price": 1.0,
    "status": "active",
    "purchased_at": "2024-01-01T00:00:00Z"
  }
}
```

```json 400 - Validation error
{
  "success": false,
  "message": "phone_number must be a valid E.164 number."
}
```

```json 409 - Already purchased
{
  "success": false,
  "message": "The requested phone number is no longer available."
}
```

```json 500 - Server error
{
  "success": false,
  "message": "An unexpected internal server error occurred."
}
```

</ResponseExample>
