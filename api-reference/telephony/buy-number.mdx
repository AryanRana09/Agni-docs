---
title: "Buy Phone Number"
openapi: "POST /api/v1/phone-numbers/buy"
description: "Purchase a phone number returned by the availability endpoint."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Body

`application/json`

#### phone_number

`string` required

Phone number in E.164 format.

#### price

`number` required

Purchase price for the number.

#### per_minute_price_inbound

`number` required

Inbound per-minute rate.

#### per_minute_price_outbound

`number` required

Outbound per-minute rate.

<RequestExample>

```json
{
  "phone_number": "+1234567890",
  "price": 2.0,
  "per_minute_price_inbound": 1,
  "per_minute_price_outbound": 1
}
```

</RequestExample>

### Response

#### 201 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.phone_number

`string` required

#### data.price

`number` required

#### data.status

`string` required

#### data.purchased_at

`string` required

ISO 8601 timestamp.

<ResponseExample>

```json 201
{
  "success": true,
  "message": "Phone number purchased successfully",
  "data": {
    "phone_number": "+1234567890",
    "price": 2.0,
    "status": "active",
    "purchased_at": "2024-03-03T16:05:25Z"
  }
}
```

</ResponseExample>
