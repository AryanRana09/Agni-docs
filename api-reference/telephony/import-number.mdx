---
title: "Import Phone Number"
openapi: "POST /api/v1/phone-numbers/import"
description: "Import an existing phone number or SIP trunk."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Body

`application/json`

#### phone

`string` required

Phone number in E.164 format.

#### type

`string` required

Phone number source type.

#### name

`string` required

Display name for this imported number.

#### termination_uri

`string`

SIP termination URI.

#### sip_username

`string`

SIP username.

#### sip_password

`string`

SIP password.

<RequestExample>

```json
{
  "phone": "+1234567890",
  "type": "custom",
  "name": "External SIP Office",
  "termination_uri": "sip:gw.example.com",
  "sip_username": "myuser",
  "sip_password": "mypassword"
}
```

</RequestExample>

### Response

#### 201 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`string` required

#### data.phone

`string` required

#### data.type

`string` required

#### data.name

`string` required

#### data.status

`string` required

#### data.created_at

`string` required

ISO 8601 timestamp.

<ResponseExample>

```json 201
{
  "success": true,
  "message": "Phone number imported successfully",
  "data": {
    "id": "uuid-string",
    "phone": "+1234567890",
    "type": "custom",
    "name": "External SIP Office",
    "termination_uri": "sip:gw.example.com",
    "sip_username": "myuser",
    "sip_password": "mypassword",
    "status": "active",
    "created_at": "2024-03-03T16:16:26Z"
  }
}
```

</ResponseExample>
