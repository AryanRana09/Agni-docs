---

### Overview

Use this endpoint to **import a phone number** you manage with another telephony provider so it can be used within your account.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `POST`  
- **Path**: `/phone-numbers/import`

### Request body

Send a JSON object with the following fields.

| Field             | Type   | Required | Description                                                        | Example                       |
| ----------------- | ------ | -------- | ------------------------------------------------------------------ | ----------------------------- |
| `phone`           | string | Yes      | Phone number to import, in E.164 format or provider-specific form. | `"+14157774444"`              |
| `type`            | string | Yes      | Type of phone number or trunk (for example, `"sip"` or `"pstn"`).  | `"sip"`                       |
| `name`            | string | Yes      | Human-readable name for the imported number.                       | `"Support Line"`              |
| `termination_uri` | string | Yes      | SIP termination URI to route outbound calls through.               | `"sip:support@example.com"`   |
| `sip_username`    | string | Yes      | SIP username used to authenticate with the provider.               | `"support-user"`              |
| `sip_password`    | string | Yes      | SIP password used to authenticate with the provider.               | `"strong-password"`           |

<RequestExample>

```bash 201 - Import phone number
curl --request POST \
  --url https://api.ravan.ai/api/v1/phone-numbers/import \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "phone": "+14157774444",
    "type": "sip",
    "name": "Support Line",
    "termination_uri": "sip:support@example.com",
    "sip_username": "support-user",
    "sip_password": "strong-password"
  }'
```

</RequestExample>

### Successful response (201)

On success, the API returns details of the imported number.

#### Top-level fields

| Field     | Type                 | Description                                   |
| --------- | -------------------- | --------------------------------------------- |
| `success` | boolean              | Indicates whether the request was successful. |
| `message` | string               | Human-readable message about the operation.   |
| `data`    | ImportedPhoneNumber  | Imported phone number details.                |

#### `ImportedPhoneNumber` object

| Field             | Type   | Description                                        | Example                       |
| ----------------- | ------ | -------------------------------------------------- | ----------------------------- |
| `id`              | string | Unique identifier for the imported number.        | `"imp_123"`                   |
| `phone`           | string | Imported phone number.                            | `"+14157774444"`              |
| `type`            | string | Type of the imported number or trunk.            | `"sip"`                       |
| `name`            | string | Human-readable name for the imported number.     | `"Support Line"`              |
| `termination_uri` | string | SIP termination URI associated with the number.  | `"sip:support@example.com"`   |
| `sip_username`    | string | SIP username used to authenticate.               | `"support-user"`              |
| `sip_password`    | string | SIP password used to authenticate.               | `"strong-password"`           |
| `status`          | string | Current status of the imported number.           | `"active"`                    |
| `created_at`      | string | ISO 8601 timestamp when the number was imported. | `"2024-01-01T00:00:00Z"`      |

</ResponseExample>
