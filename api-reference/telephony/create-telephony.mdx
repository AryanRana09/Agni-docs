---
title: "Create Telephony Config"
openapi: "POST /api/v1/create/telephony"
description: "Configure a telephony provider such as Twilio."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Body

`application/json`

#### type

`string` required

Telephony provider type, for example `twilio`.

#### owner_type

`string` required

Owner scope for this config.

#### friendly_name

`string` required

Display name for the integration.

#### client_id

`string` required

Provider account id.

#### client_secret

`string` required

Provider account secret.

<RequestExample>

```json
{
  "type": "twilio",
  "owner_type": "platform",
  "friendly_name": "Primary Twilio Account",
  "client_id": "ACXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
  "client_secret": "YYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"
}
```

</RequestExample>

### Response

#### 201 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.id

`string` required

#### data.type

`string` required

#### data.owner_type

`string` required

#### data.friendly_name

`string` required

#### data.status

`string` required

#### data.created_at

`string` required

ISO 8601 timestamp.

<ResponseExample>

```json 201
{
  "success": true,
  "message": "Telephony configuration created successfully",
  "data": {
    "id": "uuid-string",
    "type": "twilio",
    "owner_type": "platform",
    "friendly_name": "Primary Twilio Account",
    "status": "active",
    "created_at": "2024-03-03T16:16:26Z"
  }
}
```

</ResponseExample>
