---
title: "Release Phone Number"
openapi: "DELETE /api/v1/phone-numbers/{phone_number}/release"
description: "Release a purchased phone number back to the provider."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### phone_number

`string` required

Purchased phone number in E.164 format.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object` required

#### data.phone_number

`string` required

#### data.status

`string` required

#### data.released_at

`string` required

ISO 8601 timestamp.

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Phone number released successfully",
  "data": {
    "phone_number": "+1234567890",
    "status": "released",
    "released_at": "2024-03-03T16:16:26Z"
  }
}
```

</ResponseExample>
