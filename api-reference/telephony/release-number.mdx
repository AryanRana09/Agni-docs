title: "Release Phone Number"

description: "Release a previously purchased phone number."
---

### Overview

Use this endpoint to **release a phone number** that you previously purchased so it is no longer associated with your account.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `DELETE`  
- **Path**: `/phone-numbers/{phone_number}/release`

### Path parameters

| Name           | Type   | Required | Description                                      | Example           |
| -------------- | ------ | -------- | ------------------------------------------------ | ----------------- |
| `phone_number` | string | Yes      | Phone number to release, in E.164 format.       | `"+14157774444"`  |

### Successful response (200)

On success, the API returns information about the released number.

#### Top-level fields

| Field     | Type                 | Description                                   |
| --------- | -------------------- | --------------------------------------------- |
| `success` | boolean              | Indicates whether the request was successful. |
| `message` | string               | Human-readable message about the operation.   |
| `data`    | ReleasedPhoneNumber  | Released phone number details.                |

#### `ReleasedPhoneNumber` object

| Field           | Type   | Description                                          | Example            |
| -------------- | ------ | ---------------------------------------------------- | ------------------ |
| `phone_number` | string | Released phone number in E.164 format.              | `"+14157774444"`   |
| `status`       | string | Status of the number after release (for example, `"released"`). | `"released"` |
| `released_at`  | string | ISO 8601 timestamp when the number was released.    | `"2024-01-01T01:00:00Z"` |
