---
title: "List Purchased Numbers"
openapi: "GET /api/v1/list-phone-numbers"
description: "List phone numbers already purchased for your account."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Query parameters

#### region

`string`

Region or state filter, for example `NY`.

#### iso_country

`string`

Two-letter ISO country code, for example `US`.

#### is_active

`boolean`

Filter by active status.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object[]` required

#### data[].id

`string` required

#### data[].number

`string` required

#### data[].friendly_name

`string`

#### data[].region

`string`

#### data[].iso_country

`string`

#### data[].is_active

`boolean` required

#### data[].created_at

`string` required

ISO 8601 timestamp.

#### data[].updated_at

`string` required

ISO 8601 timestamp.
