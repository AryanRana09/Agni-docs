### Overview

Use this endpoint to **list phone numbers you have already purchased**, optionally filtered by region, country, and active status.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `GET`  
- **Path**: `/list-phone-numbers`

### Query parameters

| Name         | Type    | Required | Description                                                        | Example   |
| ------------ | ------- | -------- | ------------------------------------------------------------------ | --------- |
| `region`     | string  | No       | Region or area within the country (for example, state or city).   | `"CA"`    |
| `isoCountry` | string  | No       | Two-letter ISO country code to filter phone numbers by.           | `"US"`    |
| `isActive`   | boolean | No       | Filter by active status. If omitted, all numbers are returned.    | `true`    |

### Successful response (200)

On success, the API returns a list of purchased numbers.

#### Top-level fields

| Field     | Type                     | Description                                   |
| --------- | ------------------------ | --------------------------------------------- |
| `success` | boolean                  | Indicates whether the request was successful. |
| `message` | string                   | Human-readable message about the operation.   |
| `data`    | array\<PurchasedNumber>  | List of purchased phone numbers.             |

#### `PurchasedNumber` object

| Field                    | Type    | Description                                                     | Example                   |
| ------------------------ | ------- | --------------------------------------------------------------- | ------------------------- |
| `id`                     | string  | Unique identifier for the purchased number.                    | `"pn_123"`                |
| `telephony_id`           | string  | Identifier of the telephony provider configuration.            | `"tel_123"`               |
| `subscription_id`        | string  | Identifier of the subscription associated with this number.    | `"sub_123"`               |
| `inbound_trunk_id`       | string  | Identifier of the inbound trunk used for this number.          | `"in_trunk_1"`            |
| `outbound_trunk_id`      | string  | Identifier of the outbound trunk used for this number.         | `"out_trunk_1"`           |
| `number`                 | string  | Phone number in E.164 format.                                  | `"+14157774444"`          |
| `friendly_name`          | string  | Human-readable label for the number.                           | `"Support Line"`          |
| `per_minute_price_inbound`  | int32 | Per-minute price for inbound calls in minor units.             | `1`                       |
| `per_minute_price_outbound` | int32 | Per-minute price for outbound calls in minor units.            | `2`                       |
| `price`                  | int32   | One-time purchase price for the number in minor units.         | `100`                     |
| `region`                 | string  | Region or area associated with the number.                     | `"CA"`                    |
| `iso_country`            | string  | Two-letter ISO country code.                                   | `"US"`                    |
| `is_active`              | boolean | Whether the number is currently active.                        | `true`                    |
| `created_at`             | string  | ISO 8601 timestamp when the number was created.                | `"2024-01-01T00:00:00Z"`  |
| `updated_at`             | string  | ISO 8601 timestamp when the number was last updated.           | `"2024-01-02T00:00:00Z"`  |
