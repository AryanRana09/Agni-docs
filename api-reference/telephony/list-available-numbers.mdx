---
title: "List Available Numbers"
openapi: "GET /api/v1/available-numbers/{iso_country}"
description: "Search phone numbers you can purchase in a country."
---

### Authorizations

#### Authorization

`string` header `required`

Use `Bearer YOUR_API_KEY`.

### Path parameters

#### iso_country

`string` required

Two-letter ISO country code, for example `US`.

### Query parameters

#### page_size

`integer`

Number of records to return.

#### region

`string`

Region or state filter, for example `NY`.

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

#### data

`object[]` required

#### data[].iso_country

`string` required

#### data[].phone_number

`string` required

#### data[].price

`number` required

#### data[].per_minute_price_outbound

`number`

#### data[].per_minute_price_inbound

`number`
