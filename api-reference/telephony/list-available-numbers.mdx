### Overview

Use this endpoint to **list available phone numbers** for a given country, with optional filtering by region and pagination.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `GET`  
- **Path**: `/available-numbers/{iso_country}`

### Path parameters

| Name         | Type    | Required | Description                                                                 | Example   |
| ------------ | ------- | -------- | --------------------------------------------------------------------------- | --------- |
| `iso_country` | string | Yes      | Two-letter ISO country code for which to list available numbers.           | `"US"`    |

### Query parameters

| Name        | Type   | Required | Description                                                                                  | Example     |
| ----------- | ------ | -------- | -------------------------------------------------------------------------------------------- | ----------- |
| `pageSize`  | int32 | No       | Maximum number of phone numbers to return in a single page.                                  | `20`        |
| `region`    | string | No      | Region or area within the country to filter numbers by (for example, state or city prefix). | `"CA"`      |

### Successful response (200)

On success, the API returns a JSON object with a list of available numbers.

#### Top-level fields

| Field      | Type                | Description                                   |
| ---------- | ------------------- | --------------------------------------------- |
| `success`  | boolean             | Indicates whether the request was successful. |
| `message`  | string              | Human-readable message about the operation.   |
| `data`     | array\<PhoneNumber> | List of available phone numbers.             |

#### `PhoneNumber` object

| Field                       | Type   | Description                                                  | Example        |
| -------------------------- | ------ | ------------------------------------------------------------ | -------------- |
| `iso_country`              | string | Two-letter ISO country code.                                | `"US"`         |
| `phone_number`             | string | Phone number in E.164 format.                               | `"+14157774444"` |
| `price`                    | int32  | One-time purchase price for the number (in minor units).    | `100`          |
| `per_minute_price_outbound` | int32 | Per-minute outbound price for calls using this number.      | `2`            |
| `per_minute_price_inbound`  | int32 | Per-minute inbound price for calls to this number.          | `1`            |
