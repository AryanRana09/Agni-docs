### Overview

Use this endpoint to **list agents** belonging to a specific organization with basic pagination.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `GET`  
- **Path**: `/agents`

### Query parameters

| Name             | Type   | Required | Description                                      | Example    |
| ---------------- | ------ | -------- | ------------------------------------------------ | ---------- |
| `organizationId` | string | Yes      | Organization identifier to list agents for.      | `"org_123"` |
| `limit`          | int32  | No       | Maximum number of agents to return.              | `10`       |
| `offset`         | int32  | No       | Number of agents to skip for pagination.         | `0`        |

### Successful response (200)

On success, the API returns a list of agents with pagination metadata.

#### Top-level fields

| Field     | Type          | Description                                   |
| --------- | ------------- | --------------------------------------------- |
| `success` | boolean       | Indicates whether the request was successful. |
| `message` | string        | Human-readable message about the operation.   |
| `data`    | array\<Agent> | List of agents.                               |
| `meta`    | Meta          | Pagination metadata.                          |

#### `Meta` object

| Field    | Type  | Description                             | Example |
| -------- | ----- | --------------------------------------- | ------- |
| `total`  | int64 | Total number of matching agents.        | `42`    |
| `limit`  | int32 | Maximum number of agents per page.      | `10`    |
| `offset` | int32 | Offset for the current page.            | `0`     |
