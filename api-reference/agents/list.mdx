---
title: "List Agents"
openapi: "GET /api/v1/agents"
description: "List agents for an organization with pagination."
---

### Authorizations

#### Authorization

`string` header `required`

Bearer token in format `Bearer YOUR_API_KEY`.

### Query parameters

#### organization_id

`string` required

Organization id.

#### limit

`integer`

Page size.

Example: `10`

#### offset

`integer`

Page offset.

Example: `0`

### Response

#### 200 application/json

#### success

`boolean` required

#### message

`string` required

Example: `"Agents fetched"`

#### data

`object[]` required

List of agent records.

#### meta

`object` required

#### meta.total

`integer` required

#### meta.limit

`integer` required

#### meta.offset

`integer` required

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Agents fetched",
  "data": [],
  "meta": {
    "total": 1,
    "limit": 10,
    "offset": 0
  }
}
```

</ResponseExample>
