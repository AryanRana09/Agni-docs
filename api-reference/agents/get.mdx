---
title: "Get Agent"
openapi: "GET /api/v1/agents/{id}"
description: "Fetch a single agent by id."
---

### Authorizations

#### Authorization

`string` header `required`

Bearer token in format `Bearer YOUR_API_KEY`.

### Path parameters

#### id

`integer` required

Agent id.

### Query parameters

#### organization_id

`string` required

Organization id that owns the agent.

Example: `org_123`
