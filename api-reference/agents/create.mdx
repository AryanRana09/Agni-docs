### Overview

Use this endpoint to **create a new agent** with configuration for model, voice, call behavior, tools, and integrations.

The base URL for all requests from the playground is `https://api.ravan.ai/api/v1`.

### Authorization

- **Authorization**: `Bearer YOUR_API_KEY` (header, required)  
  Example: `Authorization: Bearer sk_live_xxx`

### HTTP request

- **Method**: `POST`  
- **Path**: `/agents`

### Request body

Send a JSON object with the following fields. Unless otherwise noted, all fields are optional and can be updated later.

| Field                         | Type    | Required | Description                                                              |
| ----------------------------- | ------- | -------- | ------------------------------------------------------------------------ |
| `organization_id`            | string  | Yes      | Organization identifier that owns the agent.                             |
| `widget_settings_id`         | int64   | No       | ID of associated widget settings.                                        |
| `name`                       | string  | No       | Internal name of the agent.                                             |
| `agent_name`                 | string  | No       | Display name shown to end users.                                        |
| `model`                      | string  | No       | Identifier of the model used for the agent.                             |
| `s2s_model`                  | string  | No       | Identifier of the speech-to-speech model, if used.                      |
| `voice_id`                   | string  | No       | Voice identifier for audio responses.                                   |
| `temperature`                | number  | No       | Sampling temperature for the model.                                     |
| `reminder_trigger_ms`        | int32   | No       | Milliseconds of silence before sending a reminder.                      |
| `reminder_max_count`         | int32   | No       | Maximum number of reminders to send.                                    |
| `ambient_sound`              | string  | No       | Ambient sound preset.                                                   |
| `ambient_sound_volume`       | number  | No       | Volume of ambient sound (0–1).                                          |
| `end_call_after_silence`     | boolean | No       | Whether to end the call after a period of silence.                      |
| `max_call_duration_ms`       | int32   | No       | Maximum call duration in milliseconds.                                  |
| `ring_duration_ms`           | int32   | No       | Maximum ring duration in milliseconds.                                  |
| `enable_voicemail_detection` | boolean | No       | Whether to enable voicemail detection.                                  |
| `voicemail_message`          | string  | No       | Message played when voicemail is detected.                              |
| `voicemail_detection_timeout_ms` | int32 | No    | Timeout in milliseconds for voicemail detection.                        |
| `ivr_options`                | object  | No       | IVR configuration as a JSON object.                                     |
| `post_call_analysis_model`   | string  | No       | Model used for post-call analysis.                                      |
| `analysis_summary_prompt`    | string  | No       | Custom prompt for call summaries.                                       |
| `selected_tools`             | object  | No       | Tools configuration as a JSON object.                                   |
| `integrations`               | object  | No       | Integrations configuration as a JSON object.                            |
| `knowledge_base`             | string  | No       | Identifier of the knowledge base used by the agent.                     |
| `begin_message`              | string  | No       | Initial message spoken by the agent.                                    |
| `start_speaker`              | string  | No       | Who speaks first, for example `"agent"` or `"user"`.                    |

<RequestExample>

```bash 201 - Create agent
curl --request POST \
  --url https://api.ravan.ai/api/v1/agents \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "organization_id": "org_123",
    "name": "support-agent",
    "agent_name": "Support Agent",
    "model": "gpt-4.1",
    "voice_id": "voice_123",
    "temperature": 0.7,
    "begin_message": "Hi, how can I help you today?",
    "start_speaker": "agent"
  }'
```

</RequestExample>
